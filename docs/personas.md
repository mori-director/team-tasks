# 페르소나 — 사내 일감 할당 앱 (team-tasks)

> **MVP 헌장** : 단일 테이블 · 기능 5개 이하 · 1일 안에 코드로 옮길 수 있는 분량

| 구분 | 역할 | 기술 수준 | 주요 시나리오 |
|------|------|-----------|--------------|
| **P1 팀장 지은** | 프로젝트 리드 — 일감을 만들고 팀원에게 배분 | 비개발자, 스프레드시트 중심으로 일함 | ① 월요일 오전, 이번 주 일감 5건을 한 번에 입력하고 담당자·마감일을 지정한다 ② 보드를 한눈에 보고 병목(진행 중 과다)을 파악해 재배분한다 ③ 금요일 오후, '완료' 컬럼만 훑어 주간 성과를 팀에 공유한다 |
| **P2 개발자 민준** | 실무 담당자 — 배정된 일감을 실행 | 개발자, CLI·IDE 선호 | ① 아침 첫 업무로 '할 일' 컬럼에서 자기 이름 필터를 걸어 오늘 할 일을 확인한다 ② 작업을 시작할 때 카드를 '진행 중'으로 이동시켜 팀에게 착수를 알린다 ③ PR 머지 후 카드를 '완료'로 옮기고 설명란에 PR 링크를 남긴다 |
| **P3 외부 협력사 수아** | 단기 참여 파트너 — 특정 일감만 담당 | 비개발자, 브라우저만 사용 | ① 초대 링크로 처음 접속해 로그인 없이 자신에게 배정된 카드만 빠르게 찾는다 ② 담당 카드의 마감일이 임박했을 때 상태를 '진행 중'으로 바꾸고 메모를 추가한다 ③ 납품 완료 후 카드를 '완료'로 이동해 팀장에게 별도 연락 없이 진행 상황을 공유한다 |
