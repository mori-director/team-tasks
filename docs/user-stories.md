# 유저 스토리 — team-tasks MVP

## 2. 결정 사항

페르소나 시나리오에서 도출한 미결 질문 5개입니다. 답변에 따라 기능 범위가 달라집니다.

1. **누구나 일감을 만들 수 있는가?**
   팀장(P1)만 생성 권한을 갖는가, 아니면 담당자(P2)와 협력사(P3)도 직접 일감을 추가할 수 있는가?

2. **담당자도 다른 사람에게 일감을 넘길 수 있는가?**
   배정된 일감의 담당자 필드를 본인이 수정할 수 있는가, 아니면 팀장만 재배분 가능한가?

3. **협력사(P3)는 자기 카드를 어떻게 찾는가?**
   담당자 필터를 직접 선택하는 것으로 충분한가, 아니면 `?assignee=수아` 같은 전용 URL이 있어야 하는가?

4. **설명란은 자유 텍스트인가, 구조화된 필드인가?**
   PR 링크·메모를 하나의 텍스트 박스에 자유롭게 입력하는가, 아니면 링크 전용 필드를 별도로 둘 것인가?

5. **완료된 일감은 어떻게 처리하는가?**
   '완료' 컬럼에 계속 쌓아 두는가, 일정 기간 후 숨기거나 별도 아카이브 뷰로 이동하는가?
